# AI_tuk — บันทึกงาน (Task Log)

repo นี้ใช้เก็บบันทึก **ทุกงานที่สั่งให้ Claude ทำ** เพื่อเป็นประวัติย้อนหลังและติดตามสถานะ

## โครงสร้าง

- [`TASKS.md`](TASKS.md) — ตารางสรุปงานทั้งหมด (รายการล่าสุดอยู่บนสุด)
- [`tasks/`](tasks/) — **1 งาน = 1 โฟลเดอร์** ชื่อ `YYYY-MM-DD-NN-ชื่องาน/`

แต่ละโฟลเดอร์งานมีโครงแบบนี้:

```
tasks/YYYY-MM-DD-NN-ชื่องาน/
├── README.md      ← รายละเอียดงาน (โจทย์ / สิ่งที่ทำ / สถานะ)
└── <ไฟล์ผลงาน>    ← โค้ด รูป ผลลัพธ์ ฯลฯ ที่เกิดจากงานนี้ เก็บรวมไว้ที่นี่
```

## สถานะงาน

| สถานะ | ความหมาย |
|-------|----------|
| 🟡 todo | รับงานแล้ว ยังไม่เริ่ม |
| 🔵 doing | กำลังทำ |
| ✅ done | เสร็จแล้ว |
| ⏸️ paused | พักไว้ |
| ❌ cancelled | ยกเลิก |
