# #02 — ทุกงานมีโฟลเดอร์แยกของตัวเอง

- **วันที่สั่ง:** 2026-06-23
- **สถานะ:** ✅ done

## โจทย์

> ทุกครั้งที่สร้างงาน ช่วยสร้างโฟลเดอร์แยกให้ชัดเจนทุกครั้ง

## สิ่งที่ทำ

- เปลี่ยนรูปแบบจาก "ไฟล์เดียวต่องาน" → **"โฟลเดอร์เดียวต่องาน"**
- แต่ละงานได้โฟลเดอร์ `tasks/YYYY-MM-DD-NN-ชื่องาน/` พร้อม `README.md` ข้างใน
- ย้ายงาน #01 เข้าโฟลเดอร์ของตัวเอง (`tasks/2026-06-23-01-setup-task-logging/README.md`)
- ไฟล์/ผลงานที่เกิดจากงานแต่ละชิ้น ให้เก็บไว้ในโฟลเดอร์ของงานนั้น

## โครงสร้างไฟล์ในโฟลเดอร์งาน (แนะนำ)

```
tasks/YYYY-MM-DD-NN-ชื่องาน/
├── README.md      ← รายละเอียดงาน (โจทย์ / สิ่งที่ทำ / สถานะ)
└── <ไฟล์ผลงาน>    ← โค้ด รูป ผลลัพธ์ ฯลฯ ที่เกิดจากงานนี้
```
